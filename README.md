## 插件介绍
Zero是基于BotPro的娱乐群管插件
目前插件还在开发测试阶段

## 安装方法
下载源码后，到BotPro后台上传插件 

上传后打开BotPro插件目录 app/Plugins 

确保本插件目录名为 `zero`

然后cd到 zero目录下

执行: `composer install`
